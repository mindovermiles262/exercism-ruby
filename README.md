## CLI Tooling
Linux: http://exercism.io/clients/cli/linux
MacOS: http://exercism.io/clients/cli/mac

## Fetching
`exercism fetch ruby`

## Configuring
Obtain API Key from:

http://exercism.io/account/key

then `exercism configure --key=YOUR_EXERCISM_KEY`

## Submitting
`exercism submit PATH_TO_FILE`
